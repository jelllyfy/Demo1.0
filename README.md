# Demo1.0

# Overview
This is a swift playground (iOS 18+) webkit based browser. To use it just copy and paste the code from the repo into your project files. 

# Warning 
- Some browser features might not work.
- When opening a new tab or searching something you might need to double click the tab button (known bug).
- Bookmark is not a finished feature so it might not work.
- Search engines might be not what they are labled as
- This browser will not let you bypass mdm or profile restrictions
